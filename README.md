# Chess-Game
Interactive chess game built with Python and Pygame.

The game supports drag-and-drop movement, turn logic, legal piece movement, captures, and early check detection.
I’m actively expanding the project to include full chess rules, win detection, and an AI opponent.

https://github.com/GOATofgoat/Chess-Game/issues/1#issue-3678461985



FEATURES IMPLEMENTED:
   8×8 chessboard rendering

   Drag-and-drop piece movement

   Legal movement for all standard pieces

   Piece capturing

   Turn-based game logic (White/Black)

   Basic check detection

   Board coordinate helper functions

   Organized object-oriented code structure
   

FEATURES IN PROGRESS:
  Castling (king-side & queen-side)
  
  Pawn promotion UI
  
  Checkmate & stalemate detection
  
  Move history + undo
  
  Minimax AI opponent with alpha-beta pruning
  
  Menu screen & UI improvements
